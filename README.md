OHHelper
========

To jest program rozszerzajacy "Otaku Helper". Kopiuje linki do epków z linku strony anime.

Instalacja:
Przekopiowanie exe

Uzywanie:
Wlaczamy Otaku Helper'a,
Zaznaczamy "Automatyczne dodawanie"
Kopiujemy link do anime (do anime, nie odcinka)
Wrzucamy do textboxa, klikamy "Dodaj anime"
Anime laduje w liscie, teraz jak chcemy dodac do Otaku Helpera to albo zaznaczamy i klikamy "Dodaj zaznaczone", albo klikamy "Dodaj wszystkie"
Czas przerwy jest to czas [ms], ktory jest odczekiwany po przekopiowaniu jednego linku
